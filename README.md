# rust-python-linter

A [Pyflakes](https://github.com/PyCQA/pyflakes)-inspired Python linter, written in Rust.

Supports:

- Python 3.10
- True parallelism
- [ESLint](https://eslint.org/docs/latest/user-guide/command-line-interface#caching)-inspired
  caching

```
cargo fmt
cargo clippy
cargo run resources/test/src
```
